前幾天把自架的 RSS 服務從 Miniflux 換成了 FreshRSS，最關鍵的理由是 Youlag 更新了一個本人期待很久的功能：**允許自訂在哪些分類使用影片視圖**。本來只能全局或關閉，想看文章列表就得經常手動切換。

我討厭 YouTube 的推送機制，開了小鈴鐺也不一定會推送到主頁面，只能在右上角小小的通知區翻找，還會自作主張將認為使用者最感興趣的內容不斷排到最上面。於是瀏覽那些自己最關注頻道的新影片時體驗很差，也容易因此漏掉一些影片。

離開 Folo 後，最想念的就是影片視圖功能，幸好現在能夠用 FreshRSS 搭配 Youlag 補足這部分的缺憾。

## 介紹

Youlag 是一款開源的 FreshRSS 擴充功能，為桌面端和行動端都提供了漂亮的影片視圖介面，隨著不斷更新，現在也加強了文章閱讀介面，讓整個 FreshRSS 的 UI 都顯得煥然一新，可以說兼具了主題功能。

GitHub 連結：[https://github.com/civilblur/youlag]()

## 額外調整

目前文章內的圖片沒有圓角，個人覺得有圓角和整個現代化的介面比較搭配，可啟用 FreshRSS 內建的擴充功能 `User CSS` ，並加入以下 CSS 啟用圖片圓角：

```css
.flux_content img {
    border-radius: 18px;
}
```

訂閱源有錯誤的話，會在側邊欄的分類前顯示一個紅色三角形警告，覺得礙眼可加上以下 CSS 隱藏：

```css
.category .title.error::before {
    display: none;
}
```

~~年紀大了視力不佳~~覺得預設的 <ruby>Ruby<rt>我是 Ruby</rt></ruby> 字體太小，可以加上以下 CSS 放大：

```css
rt {
    font-size: 0.7rem;
}
```

## 結語

Youlag 真的很讚，當初會選擇 Miniflux，滿大一個原因是找不到讓 FreshRSS 的介面足夠現代化的主題，我偏好圓潤並有適度留白的設計。雖然 Miniflux 搭配 ReactFlux 這樣的第三方前端，在動畫的部分依然更精緻一點，但要調整一些細部設定，還是得常常回到 Miniflux，有點像廁所不在房間裡的感覺，不太方便。

這個擴充功能不只將外觀提升到個人能接受的水準，還提供了很不錯的影片視圖。有時候真的會忘記自己不是在 YouTube，直到發現沒有👍可以按 lol 可惜 FreshRSS 沒有子分類的功能，不然就更理想了。這樣可以完全替代 YouTube 的訂閱，然後用分類整理訂閱的頻道，現在想這麼做的話，和文章的分類在一起會讓分類變得太多，或者得開一個獨立的帳號專門管理影片的 RSS。
